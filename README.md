# infra-code

Personal dev infrastructure monorepo, primarily focused on ESLint configuration.

## Packages

| Package | Description |
|---------|-------------|
| [`@infra-x/eslint-config`](./packages/eslint-config) | Composable ESLint config |
| [`@infra-x/typescript-config`](./packages/typescript-config) | Shared TypeScript config |
| [`@infra-x/create-eslint-config`](./packages/create-eslint-config) | CLI scaffolding tool |

## Getting Started

```bash
pnpm dlx @infra-x/create-eslint-config
```

Or refer to the README in each package directory.
