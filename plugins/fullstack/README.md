# Fullstack Plugin

全栈开发工具集，整合前后端工作流。

## 功能

### Commands
- `/project` - 全栈项目初始化
- `/feature` - 端到端功能开发
- `/deploy` - 部署流程
- `/connect` - 前后端连接

### Agents
- `fullstack-architect` - 全栈架构师
- `deployment-expert` - 部署专家

### Skills
- `api-connector` - API 集成
- `project-initializer` - 项目初始化

## 使用

```bash
/project my-app
/feature user-auth
/deploy production
```
