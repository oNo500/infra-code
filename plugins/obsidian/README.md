# Obsidian Plugin

Obsidian 知识库管理工具集。

## 功能

### Commands
- `/note-create` - 创建新笔记
- `/note-search` - 搜索笔记
- `/note-link` - 链接管理
- `/note-sync` - 同步整理

### Agents
- `knowledge-organizer` - 知识组织专家
- `note-summarizer` - 笔记摘要生成

### Skills
- `note-linker` - 自动双向链接
- `markdown-formatter` - Markdown 格式化

## 使用

```bash
/note-create 我的新笔记
/note-search 关键词
```
