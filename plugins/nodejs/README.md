# Node.js Plugin

Node.js 后端开发工具集。

## 功能

### Commands
- `/api` - RESTful API 路由生成
- `/model` - 数据模型生成
- `/middleware` - 中间件生成
- `/test-api` - API 测试

### Agents
- `api-designer` - API 设计专家
- `security-auditor` - 安全审计专家

### Skills
- `error-handler` - 错误处理生成
- `validation-builder` - 验证逻辑生成

## 使用

```bash
/api /users
/model User
```
