# CLI DX Plugin

脚手架和开发体验提升工具集。

## 功能

### Commands
- `/scaffold` - 项目脚手架生成
- `/template` - 模板管理
- `/config` - 配置向导
- `/upgrade` - 项目升级

### Agents
- `template-designer` - 模板设计专家
- `dx-optimizer` - DX 优化顾问

### Skills
- `config-generator` - 配置生成
- `template-renderer` - 模板渲染

## 使用

```bash
/scaffold react-app my-project
/template list
/config
```

## 模板

将项目模板放在 `templates/` 目录下。
