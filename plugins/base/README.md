# Common Plugin

通用工具集，提供常用的便捷命令和个人定制功能。

## 功能

### Commands
- `/hello` - 友好问候

### MCP Servers
- **filesystem** - 文件系统访问
- **context7** - 库文档查询（获取最新的库文档和代码示例）

## 定位

这是一个通用插件，用于存放：
- 通用的便捷命令
- 个人工作流定制
- 跨领域的实用工具
- 实验性功能测试

## 使用

```bash
# 使用命令
/hello

# Context7 会自动集成到工具集
# 可以直接查询任何库的最新文档
```

## 扩展

你可以在此插件中添加任何通用的命令、代理或技能。
