# Web Dev Plugin

现代前端开发工具集。

## 功能

### Commands
- `/component` - React/Vue 组件生成
- `/page` - 页面生成
- `/style` - 样式文件生成
- `/build` - 构建优化

### Agents
- `ui-reviewer` - UI/UX 审查专家
- `performance-optimizer` - 性能优化专家

### Skills
- `component-generator` - 智能组件生成
- `css-optimizer` - CSS 优化

## 技术栈支持

- React / Next.js
- Vue / Nuxt
- Tailwind CSS
- Vite / Webpack

## 使用

```bash
/component Button
/page /dashboard
```
